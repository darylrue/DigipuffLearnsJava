How to Create a Jar File in IntelliJ IDEA

Once your program works and you are ready to create a Jar file:

1. Click File, Project Structure, then on the left side click Artifacts.

2. Click the green Plus symbol, then select JAR, then From modules with dependencies...

3. Beside Main Class, click the button with the 3 dots.

4. Click the Project tab, then expand your project folder, then src, then your package, then select your program and click OK.

5. Beside Directory for META-INF/MANIFEST.MF:, add a folder to the path for your manifest file. For example:

change: 

C:\Users\2459037845\Documents\Java\Workspace\HelloWorld\src

to: 

C:\Users\2459037845\Documents\Java\Workspace\HelloWorld\src\HelloWorldMF

6. Click OK.

7. In the top field beside Name:, change the name of the artifact, if desired. This will be the name you select when you want to build the .jar file.

8. In the middle pane, just above the META_INF folder, you will see the filename of your jar. If you wish to change it, right click it and select Rename. Change the name and then press ENTER.

9. Click OK.

Now, back on the main screen of IntelliJ:

10. Click Build, then Build Artifacts, select your artifact name, then click Build.

Once it is finished, your Jar file will be in the out/artifacts/[artifact name]/ folder.

12. Right click your Jar file and select Run to make sure it works.
