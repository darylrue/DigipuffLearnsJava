How to Add Haikus

Haikus appear when students successfully complete challenges, and are chosen at random from the file haikus.txt, located in the /res folder.

To add haikus to the file, make sure to use the <haiku> tag on the first line, place the haiku on the next three lines, and then place the </haiku> tag on the last line.

Do not try to add comments to the file, use line spaces between haikus, or use a different number of lines for the haiku, as all of these things will trigger an error. An example of a valid haiku is below:


<haiku>
This is my haiku
And it has a second line
Thank you, please drive through
</haiku>
